Project Details

College course: CS492 Computer Science Team Project Part 2

In a group size of four, we were assigned to automate a Book Store using the Scrum framework.

Project Requirements:

-Keeping track of inventory

-Keeping track of sales records

-Providing the abilities to place orders to manufacturers

-Providing the ability for customers to place orders


Technologies

Communication Method: Discord

IDE & Coding Sessions: IntelliJ

Product Backlog: Google Sheets

Product Doc: Google Docs

Frontend: HTML, CSS, Javascript

Backend: Java, Spring Framework, H2 Database
