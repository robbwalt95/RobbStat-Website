History - Purpose

This site was created in 2015.

At creation, the purpose was to be a personal blog.

Eventually the site was reshaped and constructed to display my personal projects, hobbies, and literature.


Technologies

-Domain was gathered from freenom.

-Connecting the domain and hosting is by 000Webhost

-FTP to upload files is done through FileZilla

-Content and functionality written in standard HTML, CSS, and JavaScript

-Files organized, edited, and tested within the IDE PyCharm

-SSL Certification gathered from CloudFlare


Features

-Simple interface with easy navigation

-Accessible through the internet

-Displayable in Desktop and Mobile view

-Provides security protocol encryption with SSL

-Accessible files such as documents and mp3s
