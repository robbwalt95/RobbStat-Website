Purpose

Seconds before planning and developing this project I realized the Music section was looking a bit lonely.

Not being a skilled musician, I decided: "Hey let's share my music playlist", thus becoming the start of this project.


Technologies

-Mp3tag Application

-Python - TinyTag library

-JSON

-Javascript - Fetch API

-HTML


Implementing Flow

Task 1 - Converting Music Metadata into JSON Format

1. Gather all music files and put them into an accessible directory

2. Using Mp3tag check each file and make sure metadata (Title and Artist) exists

3. In Python write to the top of the file with the heading format

4. Using Python access the music directory and iterate through each file

5. With each file iteration exact the metadata (TinyTag) and append it in with the correct format for JSON

6. Append the closing format to the JSON file


Task 2 - Displaying JSON File into HTML Web Page

1. Analyzed THIS video

2. Literally followed directions to implement the html table and the Javascript Fetch API script

3. Reviewed what is written and checked for silly made errors

4. Confirmed the results on a server


Author Note of Development

The development of this project was quick. However, there were some hurdles to jump through before it got to satisfactory standards.

The first hurdle was collecting flac file metadata. I used the eye3d python library only to find I could only properly gather from the mp3 format.

I switched over to the Mutagen python library. Ultimately, I found that this library could not get m4a file metadata.

The TinyTag python library was my final solution to read metadata from file formats of: mp3, flac, and m4a.


Task 2 was pretty straightforward. Only major issues I had to deal with were from syntax and variable names.

The video I followed (Thank you dcode) was a great template that converted into my project easily.
